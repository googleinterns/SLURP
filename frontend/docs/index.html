components/Utils/temp-auth-utils.js

This is a temporary file that is used to implement 'fake' versions of the Auth utility functions used in the ViewTrips components. As of right now, the only 'real' auth function used is getCurUserEmail() but a fake user uid is still used due to the lack of a set of functions that can convert user email <--> user uid.
