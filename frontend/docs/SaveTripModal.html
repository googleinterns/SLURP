Class: SaveTripModal

SaveTripModal()

Component corresponding to the save trips modal.

This component acts as a 'pseudo-parent' of the AddTripModal and EditTripModal components. The only differences in the implementation between the two fake components are dervied from the props tripid and defaultFormData (see below). The primary difference between the add and edit trip modals is the former displays placeholder values in the empty form fields whereas the latter displays the current values of the trip in the respective form fields.
